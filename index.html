HTML

  • The head element contains information about the webpage.
  • The body element represents the visible ontent shown to the user.
  • Describes the structure of a web page.
  • Tells the browser how to display the content.
  • Label pices of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

CSS

  • CSS stands for Cascading Style Sheets.
  • Control many style properties of an element.
  • Coloring
  • Size
  • Position
  • Visibility
  • Used in animation

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript
